RS232 Serial Logger

The Serial Logger (_rc01) has these features

  • builds with Visual Studio 2017
  • runs on 32-bit Windows 10
  • logs all the serial port data to auto-named files
  • COMM1 starts with: 'dt5.exe -A1 -B9600' for example
